Étape 4 : Le cerveau (JavaScript)
C'est ici que la magie opère pour faire les calculs. Copie ceci dans la partie JS :

JavaScript
function calculer() {
    // On récupère les valeurs saisies
    let bio = parseFloat(document.getElementById('bio').value) || 0;
    let serviettes = parseFloat(document.getElementById('serviettes').value) || 0;
    let emballages = parseFloat(document.getElementById('emballages').value) || 0;
    let fruits = parseFloat(document.getElementById('fruits').value) || 0;
    let pain = parseFloat(document.getElementById('pain').value) || 0;

    // Calculs
    let total = bio + serviettes + emballages + fruits + pain;
    let parEleve = (total / 700) * 1000; // Résultat en grammes par élève

    // Affichage
    document.getElementById('resultat').innerHTML = `
        <h3>Résultats du jour :</h3>
        <p><strong>Total des déchets :</strong> ${total.toFixed(2)} kg</p>
        <p><strong>Moyenne par élève :</strong> ${parEleve.toFixed(0)} grammes</p>
    `;
}

comment heberger ce texte

C'est une excellente question ! Une fois que ton code fonctionne sur ton ordinateur (ou sur un éditeur en ligne), l'étape de l'hébergement consiste à le mettre sur un "serveur" pour que tout le monde puisse y accéder avec une adresse (URL) type https://mon-projet-cantine.com.

Puisque tu es débutant, voici les trois solutions les plus simples, de la plus rapide à la plus "pro".

1. La solution "Zéro technique" : CodePen
Si tu as utilisé CodePen pour tester ton code, tu n'as rien à faire !

Une fois ton code collé, clique sur "Save".

L'adresse dans la barre de ton navigateur (ex: codepen.io/ton-nom/pen/abcde) est déjà un lien public.

Avantage : Instantané.

Inconvénient : L'adresse est un peu compliquée et il y a l'interface de CodePen autour.

2. La solution "Pro & Gratuite" : GitHub Pages
C'est la méthode préférée des développeurs. C'est gratuit et très propre.

Crée un compte sur GitHub.com.

Crée un nouveau "Repository" (un dossier de projet) nommé cantine-odd.

Dépose ton fichier dedans (il doit impérativement s'appeler index.html).

Va dans les Settings (Paramètres) > Pages.

Sous "Build and deployment", choisis la branche main et clique sur Save.

Résultat : Ton site sera en ligne à l'adresse https://ton-pseudo.github.io/cantine-odd/.

3. La solution "Collège" : L'ENT ou le site de l'établissement
Puisque c'est un projet scolaire, c'est souvent la meilleure option pour la visibilité :

L'ENT (MonCollège en Vaucluse) : Tu peux parfois créer des "Pages" ou des "Blogs". Tu peux y copier ton code HTML (souvent via un bouton < > ou "Source" dans l'éditeur de texte).

Le site du collège : Demande à ton professeur documentaliste ou au référent numérique. Ils peuvent héberger ton fichier sur le serveur du collège.
